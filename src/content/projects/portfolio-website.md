# Portfolio Website

This is my personal portfolio website built with Astro, a modern static site generator that delivers exceptional performance. The site showcases my projects, experience, and technical skills in cybersecurity and web development.

## Features

- **Modern Design**: Clean, responsive design that works across all devices
- **Performance Optimized**: Built with Astro for fast loading times and optimal SEO
- **Theme Support**: Multiple color themes including dark mode
- **Interactive Elements**: Smooth animations and hover effects
- **Accessibility**: Built with accessibility best practices in mind

## Technical Implementation

The website uses Astro's component-based architecture with:
- Static site generation for optimal performance
- Tailwind CSS for styling with custom theme variables
- Responsive design principles
- Modern JavaScript for interactive elements

This project demonstrates my ability to create modern, performant web applications while maintaining clean, maintainable code.
